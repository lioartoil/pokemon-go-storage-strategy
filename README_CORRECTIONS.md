**My Original Error**: Stated lucky trade cap was at 1 year

### 5. Overlap Handling ✅ CLARIFIED

**Question**: How are Pokemon counted if they qualify for multiple categories?

**Answer**: Priority hierarchy - count as 1 Pokemon in highest priority category:

1. Categories 1-3 (League competitive) - HIGHEST
2. Categories 4-10 (Special variants)
3. Categories 11-13 (Reserves)
4. Category 14 (Trade queue) - LOWEST

**Example**: Shiny Shadow Charizard with rank #5 for Great League

- Qualifies for: Category 1, 4, 6
- Counts as: **1 Pokemon in Category 1** (highest priority)
