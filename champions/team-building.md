# Pokemon Champions — Team Building Guide

> **Researched**: 2026-04-14 | **Sources**: Smogon, Champions Lab, Game8, The Click, community guides
> **Game Release**: April 8, 2026 | **Meta**: Regulation M-A (April 8 – June 17, 2026)

## Game Overview

Pokemon Champions is a free-to-play competitive battle game by The Pokemon Works (TPC + ILCA). Nintendo Switch / Switch 2, with mobile (iOS/Android) coming June 2026. Battle-only — no story, no catching. Now the official VGC platform starting 2026 Worlds.

## Core Mechanics (Key Differences from Mainline)

| Mechanic | Pokemon Champions | Mainline (SV) |
|---|---|---|
| **Stats** | 66 Stat Points, 32 cap/stat, 1 SP = +1 stat at Lv.50 | IVs (0-31) + EVs (0-252, 510 total) |
| **Level** | Fixed Lv.50 always | Variable |
| **Items** | Item Clause (no duplicates). **No** Choice Band/Specs, Life Orb, Heavy-Duty Boots, Toxic/Flame Orb | All items available |
| **Mega Evolution** | Yes (1 per battle per player) | Gen 6-7 only |
| **Terastallization** | Coming in future update | Yes |
| **Team source** | Pick 1 of 10 starters → get full 6-mon team, customize with VP | Catch/breed/trade |

**Common stat spread**: 32 + 32 into two primary stats (e.g. Atk + Spe for sweepers), 2 leftover into HP or a defense.

**VP (Victory Points)**: Earned from battles. Used to buy moves, abilities, stat resets, and new team members. Free to remove SPs; 5 VP per 1 SP to assign.

## Singles Meta (Smogon Champions OU)

### Format Differences

- 6v6 singles (modded in; Champions itself only supports 3v3 BSS natively)
- **No** Choice Band, Choice Specs, Life Orb, Heavy-Duty Boots, Toxic Orb, Flame Orb
- Item Clause (one of each item per team)
- Mega Evolution available — defines team anchor
- PP nerfs: many moves now 8 or 12 PP (PP wars relevant)
- No Gholdengo — Defog works freely
- New sleep mechanics: 1-2 turns (1/3 wake turn 1, guaranteed turn 2)

### Bans

- **Banned**: Mega Gengar (Shadow Tag + 170 SpA + Nasty Plot), Shed Tail, Moody, Baton Pass, Last Respects, Bright Powder
- **Under watch**: Mega Blastoise (Shell Smash), Espathra (Speed Boost), Palafin-Hero

### Viability Highlights

**Meta-defining Megas**:
- Mega Chesnaught — strongest defensive presence, walls Mega Excadrill
- Mega Lopunny (205 Spe) — SD + Mach Punch, Fake Out utility
- Mega Lucario (180 Spe) — SD + Bullet Punch, "busted" Fighting-type
- Mega Starmie (189 Spe) — near-uncheckable coverage (Psychic/Water/Ice/Electric)
- Mega Greninja (213 Spe) — one of the fastest, great coverage
- Mega Scizor — priority + bulk

**Non-mega staples**:
- Kingambit — Sucker Punch priority, endgame cleaner
- Dragapult (213 Spe) — fastest non-mega
- Corviknight — Defog + Will-o-Wisp pivot
- Rotom-Wash — Sitrus Berry + Will-o-Wisp, best pivot
- Sneasler (189 Spe) — Dire Claw + Close Combat
- Garchomp, Volcarona, Excadrill

### Recommended Frameworks

**Bulky Offense (most stable)**:
- 1 Mega (Chesnaught / Lopunny / Scizor)
- Kingambit (Sucker Punch priority)
- Corviknight (Defog + Will-o-Wisp)
- Rotom-Wash (secondary pivot)
- Dragapult or fast attacker
- Flex: Sneasler / Garchomp / H-Samurott

**Hyper Offense**:
- Lead: H-Samurott (Ceaseless Edge Spikes) or Glimmora
- Mega: Lopunny / Greninja / Lucario
- Choice Scarf revenge killer
- Priority: Kingambit or Palafin
- Cleanup: Volcarona (Quiver Dance)

**Rain Offense**:
- Pelipper (Drizzle)
- Mega Greninja or Basculegion (Swift Swim)
- Palafin (Jet Punch + boosted Water)
- Hurricane / Thunder users

**Sun Offense**:
- Mega Charizard Y
- Venusaur (Chlorophyll)
- Scovillain (Chlorophyll)
- Ceruledge

### Metagame Trends (April 2026)

1. **Offense dominates** — Mega power + no Band/Specs creates item parity but offense wins
2. **Priority is king** — Sucker Punch, Mach Punch, Bullet Punch, Aqua Jet, Jet Punch see heavy play
3. **No HDB** — hazards are much more punishing; Corviknight Defog is premium
4. **Fighting types critical** — Kingambit everywhere, teams need reliable Fighting answers
5. **Defensive cores are thin** — hard stall not viable, meta skews Bulky Offense
6. **Mega slot = team anchor** — without Specs/Band/Orb, Megas define strategy
7. **Primarina rising** — Moonblast very free after Mega Gengar ban

## Team Building Best Practices

### The Seven-Step Process (Community Consensus)

1. **Pick a strategy** — weather, Trick Room, balanced, HO
2. **Assign a role to every slot** — if you can't explain what it does, cut it
3. **Build type coverage** — never stack the same weakness on 3+ members
4. **Design for synergy** — weather + ability combos, pivoting chains, support enabling sweepers
5. **Use the tier list** — check Pikalytics / Champions Lab for current usage
6. **Balance speed + priority** — at least one fast mon or priority user + speed control
7. **Test and iterate** — team building is never one-draft-done

### Role Slots (Doubles)

| Slot | Function | Examples |
|---|---|---|
| Fake Out user | Disrupts turn 1 | Incineroar, Sneasler |
| Speed control | Tailwind or Trick Room | Whimsicott, Farigiraf |
| Protect user | On every sweeper | — |
| Support | Helping Hand / Coaching | Incineroar, Maushold |
| Weather setter | If running weather | Pelipper, Tyranitar, Torkoal |
| Win condition | Setup sweeper or nuke | Archaludon, Basculegion, Excadrill |

### Role Slots (Singles — Smogon)

| Role | Purpose | Key Trait |
|---|---|---|
| Wallbreaker | Punches holes, no setup needed | Immediate power |
| Setup Sweeper | Boosts stats, then cleans | SD / DD / QD / CM |
| Wall | Absorbs damage, must have recovery | Recover / Roost |
| Pivot | Forces switches, generates momentum | Volt Switch / U-turn |
| Cleaner / Revenge Killer | Finishes weakened teams | Choice Scarf or priority |
| Hazard Setter | Stealth Rock (mandatory) | — |

### Canonical Type Cores

| Core | Types | Use Case |
|---|---|---|
| **FWG** | Fire / Water / Grass | Defensive backbone, oldest and safest |
| **SFD** | Steel / Fairy / Dragon | Modern dominant, 9+ resistances combined |
| **FDP** | Fighting / Dark / Psychic | Offensive — each removes the others' counters |

### Common Mistakes

1. **Picking favorites with no role** — every slot must have a job
2. **Stacking type weaknesses** — 3+ members weak to same type = auto-lose
3. **Ignoring speed tiers** — if nothing outspeeds the meta, any Scarf sweeps you
4. **No defensive anchor** — even HO needs Fake Out / Intimidate pivot
5. **All physical or all special** — one wall stops your entire offense
6. **No win condition** — utility Pokemon with no path to victory
7. **No hazard control (Singles)** — need SR setter AND removal
8. **Building to counter everything** — scattered teams lose to focused ones
9. **Mediocre "do everything" Pokemon** — each slot must excel at one thing
10. **Not adapting** — meta is 6 days old and shifting daily
