# Pokemon GO Storage - Quick Reference Guide

**Quick lookup for daily storage decisions**

---

## When Catching a New Pokemon: Decision Tree

```
NEW POKEMON CAUGHT
    ↓
Is it Shiny/Costumed/Gigantamax/Dynamax/Special Background?
├─ YES → Check if you have < 2 of this form → KEEP if yes
└─ NO → Continue
    ↓
Is it Shadow?
├─ YES → Check if you have < 2 Shadow of this species → KEEP if yes
└─ NO → Continue
    ↓
Is it XXS or XXL (extreme size)?
├─ YES → Check if it's more extreme than current → KEEP if yes
└─ NO → Continue
    ↓
Is it Legendary/Ultra Beast?
├─ YES → Is it in Top 10 by Level+IV? → KEEP if yes
└─ NO → Continue
    ↓
Check PvP IV Ranks:
├─ Great League (CP ≤1500): Rank ≤205?
├─ Ultra League (CP ≤2500): Rank ≤205?
├─ Little League (CP ≤500): Rank ≤19 OR (Rank ≤205 AND 60th+ percentile)?
├─ Master League (Any CP): Rank ≤19?
│   ├─ YES to any → KEEP based on rank rules
│   └─ NO to all → Continue
└─ Is it in Top 2 by Level+IV for this species?
    ├─ YES → KEEP
    └─ NO → Is it 364+ days old?
        ├─ YES → KEEP for Lucky Trading (until 3 years)
        └─ NO → TRANSFER
```

## Quick Retention Rules by Category

| If Pokemon is...                    | Keep How Many?          | Sort By    |
| ----------------------------------- | ----------------------- | ---------- |
| **Rank 1-10** for GL/UL/LL          | 3 best                  | IV Rank    |
| **Rank 11-205** for GL/UL           | 1 best                  | IV Rank    |
| **Rank 11-19** for ML (final evo)   | 1 best\*                | IV Rank    |
| **Rank 11-19** for ML (not final)   | 2 best                  | IV Rank    |
| **Rank 11-205** for LL (60th+ only) | 1 best                  | IV Rank    |
| **Shiny** (any form)                | 2 best                  | IV Rank    |
| **Shadow** (any)                    | 2 best                  | IV Rank    |
| **Costumed** (per costume)          | 2 best                  | IV Rank    |
| **Background** (per background)     | 2 best                  | IV Rank    |
| **Gigantamax**                      | 2 best                  | IV Rank    |
| **Dynamax**                         | 2 best                  | IV Rank    |
| **XXS size**                        | 1 most extreme          | Size       |
| **XXL size**                        | 1 most extreme          | Size       |

## Common Mistakes to Avoid

- ❌ **Don't transfer** Pokemon with rank ≤10 for any league without checking thoroughly
- ❌ **Don't transfer** Shiny/Costumed/Shadow before checking if you have 2+
- ❌ **Don't transfer** 3-year-old Pokemon without attempting Lucky Trades first
- ❌ **Don't transfer** Mythical Pokemon during non-event times (wait for 2× candy)
- ❌ **Don't assume** 100% IV is best for Great/Ultra/Little League (it's often NOT)

- ✅ **Do transfer** excess Legendary/UB during 2× candy events
- ✅ **Do trade** 3-year-old Pokemon with Best/Ultra friends for Lucky chance
- ✅ **Do check** Little League percentile before keeping rank 20-205
- ✅ **Do verify** XXS/XXL is more extreme than current before transferring old one

## Species Separation Rules

These variants count as **SEPARATE SPECIES** for retention:

| Base      | Shadow           | Mega           | Shiny | Costumed | Background | Dynamax | Gigantamax |
| --------- | ---------------- | -------------- | ----- | -------- | ---------- | ------- | ---------- |
| Charizard | Shadow Charizard | Mega Charizard | ✓     | ✓        | ✓          | ✓       | ✓          |

**Example**: You can keep 3× rank #1 Charizard for Great League **AND** 3× rank #1 Shadow Charizard for Great League (separate species).

## IV Rank vs IV Percentage

**IMPORTANT**: IV Rank ≠ IV Percentage

- **IV Percentage**: Measure of total stats (ATK+DEF+HP) / 45

  - Example: 15/15/15 = 100%

- **IV Rank**: Measure of PvP optimization for specific CP cap
  - Example for Great League: 0/15/15 might be Rank #1
  - Example for Master League: 15/15/15 is Rank #1

**For Capped Leagues (Little/Great/Ultra)**: Lower ATK is often better (maximizes bulk)
**For Master League**: Higher total stats are better (15/15/15 ideal)

## Emergency Space Recovery (If <100 Free Slots)

Priority order for quick space recovery:

1. **Transfer non-qualifying catches** from recent days
2. **Clear Category 12** (Legendary transfer queue) during 2× event
3. **Tighten Category 3** (Master League): Reduce rank 11-19 → only keep rank ≤10
4. **Tighten Category 1** (Great/Ultra): Reduce rank 11-205 cutoff → rank 11-100
5. **Tighten Category 13** (General Reserve): Reduce from 2 → 1 per species
6. **Review Category 11** (Legendary Reserve): Reduce from 10 → 5 per species
7. **Consider transferring** Category 14 (Lucky Trade candidates) if age <2 years
